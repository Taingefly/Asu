B — Pemeriksaan Respon

AVPU — AWAS / SUARA / NYERI / TIDAK RESPON
📋
"SAYA PERIKSA RESPON KORBAN"
Kode Cara Ucapkan
A Lambaikan tangan "Dek/Kak, tangan saya terlihat?"
V Panggil korban "Dek/Kak, suara saya terdengar?"
P Tekan ulu hati "Dek/Kak, ini terasa sakit?"
U — Tidak ada respon sama sekali
Hasil: Korban dinyatakan → RESPON
💓

C — Nadi, Napas & Perfusi

PEMERIKSAAN VITAL
1
NADI
Periksa nadi Radialis (korban sadar) — hitung 1, 2, 3, 4, 5
→ Nadi teraba / tidak teraba
2
NAPAS
Simpan 2 jari di bawah hidung (korban sadar) — hitung 1, 2, 3, 4, 5
→ Napas ada / tidak ada
3
PERFUSI / KONTROL PENDARAHAN
Tekan ibu jari korban — hitung 1, 2, 3, 4, 5
⏱ > 2 Detik
Terjadi gangguan sirkulasi / pendarahan
⏱ < 2 Detik
Sirkulasi normal
4
TENANGKAN KORBAN
"Tenang ya Dek/Kak, luka sedang kami tangani. Jangan khawatir."
